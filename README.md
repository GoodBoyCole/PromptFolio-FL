# PromptFolio-FL\n\nWelcome to the PromptFolio-FL repository! This project implements federated learning techniques using Vision-Language foundation models.\n\n## Overview\nPromptFolio-FL enables theoretical analysis and provides methods for federated learning applications, offering cutting-edge techniques supported with rich examples and configurations.\n\n## Installation\n\nThe following instructions help you set up PromptFolio-FL:\n\n```bash\nconda create -n PromptFolio python=3.8 yacs tqdm tabulate ftfy regex tensorboard\nconda activate PromptFolio\npip install torch==2.1.0 torchvision==0.16.0 torchaudio==2.1.0 --index-url https://download.pytorch.org/whl/cu121\npip install timm gdown prettytable scikit-learn einops \n\ngit clone https://github.com/GoodBoyCole/PromptFolio-FL.git\ncd PromptFolio-FL\n```\n\nPlease ensure your Torch version corresponds to CUDA 12.3, as it is the tested version.\n\nConfigure your dataset directory path, e.g., `/storage/data`, using the argparse option `--root` within `federated_main.py`. For detailed dataset setup, refer to [DATASETS.md](DATASETS.md).\n\n## Running the Code\n\nHere are some examples: \n\n### Example: Run PromptFolio with Caltech101\n```bash\npython federated_main.py --trainer "configs/datasets/caltech101.yaml" --model fedavg --trainer PromptFolio --frac_p 0.4 --num_users 10 --beta 0.3 --round 10 \n``` \n\n### Example: Run the baselines with DTD\n```bash\n# python federated_main.py --dataset-config-file "configs/datasets/dtd.yaml" --num_users 10 --beta 0.3 --round 10 --model local --trainer PromptFL # CoOp\n# python federated_main.py --dataset-config-file "configs/datasets/dtd.yaml" --num_users 10 --beta 0.3 --round 10 --model fedavg --trainer PromptFL # PromptFL\n# python federated_main.py --dataset-config-file "configs/datasets/dtd.yaml" --num_users 10 --beta 0.3 --round 10 --model fedavg --trainer PromptFLFT # PromptFL+FT\n# python federated_main.py --dataset-config-file "configs/datasets/dtd.yaml" --num_users 10 --beta 0.3 --round 10 --model fedavg --trainer PromptFL --fedprox_mu 1.0 # PromptFL+FedProx (e.g., with mu=1.0)\n# python federated_main.py --dataset-config-file "configs/datasets/dtd.yaml" --num_users 10 --beta 0.3 --round 10 --model fedavg --trainer PromptFLFedPer\n# python federated_main.py --dataset-config-file "configs/datasets/dtd.yaml" --num_users 10 --beta 0.3 --round 10 --model fedavg --trainer PromptFLFedAMP\n# python federated_main.py --dataset-config-file "configs/datasets/dtd.yaml" --num_users 10 --beta 0.3 --round 10 --model fedavg --trainer FedTPG\n# python federated_main.py --dataset-config-file "configs/datasets/dtd.yaml" --num_users 10 --beta 0.3 --round 10 --model fedavg --trainer pFedPrompt\n```
